# 청년교구 여름수련회 NFC 키링 페이지 — 설계 문서

날짜: 2026-07-08
상태: 사용자 설계 승인 완료

## 목적

청년교구 여름수련회(주제: 기도) 기념 NFC 태그 키링을 찍으면 열리는 모바일 웹페이지.
주제 말씀, 포스터 갤러리, 교제내용을 한곳에 모아 제공한다.

## 요구사항

- 콘텐츠: 주제 말씀 + 포스터 갤러리 + 교제내용을 **한 페이지**에서 접근
- 업데이트: **자주 업데이트**되며, **비개발자**(교역자·리더)도 교제내용을 수정할 수 있어야 함
- 비용: **무료**
- URL: NFC 태그에 한 번 기록하면 변경 불가 → **1년 이상 안정적으로 유지**되어야 함
- 사용 환경: NFC 스캔 = 100% 모바일 → 모바일 우선 디자인

## 선택한 접근: 정적 랜딩 + 노션 하이브리드

검토한 대안: (1) 노션 공개 페이지 단독 — 디자인 자유도 부족, (2) 순수 정적 페이지 —
비개발자 수정 불가. 하이브리드가 두 요구를 모두 충족하여 선택.

### 아키텍처

```
NFC 키링 ──▶ https://<계정>.github.io/pray  (GitHub Pages, URL 불변)
                │
                ├─ 히어로: 수련회 타이틀 + 주제 말씀
                ├─ 말씀 카드: 말씀 전문 (타이포그래피 중심)
                ├─ 포스터 갤러리: 스와이프 캐러셀 (이미지는 리포에 저장)
                └─ [교제내용 보기] ──▶ 노션 공개 페이지 (비개발자가 직접 수정)
```

### 구성 요소

| 구성 요소 | 기술 | 수정 방법 |
|---|---|---|
| 랜딩 페이지 | 순수 HTML/CSS/JS 1장, 프레임워크 없음 | 바이브코딩 후 git push 재배포 |
| 호스팅 | GitHub Pages (무료, 무기한) | — |
| 교제내용 | 노션 공개 페이지 (무료 플랜) | 노션에서 직접 수정, 즉시 반영 |
| 포스터 이미지 | 리포 내 `images/` (웹용 압축) | 이미지 추가 후 재배포 |

### 디자인 방향

- 기도 테마: 차분한 배경(밤하늘/은은한 그라데이션), 은은한 애니메이션
- 말씀은 묵상하기 좋게 큰 타이포그래피
- 시스템 폰트 또는 무료 한글 웹폰트(Pretendard 등) 셀프호스팅 — 외부 CDN 의존 최소화

### NFC 실무

- 태그: NTAG213(용량 144B, URL 충분) 또는 NTAG215
- 기록: "NFC Tools" 무료 앱으로 NDEF URI 레코드 기록
- 배포 URL 최종 확인 후 쓰기 잠금(lock) 권장 — 덮어쓰기 사고 방지
- URL 단축 서비스(bit.ly 등)는 사용하지 않음 — 외부 서비스 의존은 1년+ 안정성 리스크

### 에러 처리 / 엣지 케이스

- 노션 페이지가 비공개로 바뀌면 교제내용 링크가 죽음 → 노션 "웹에 공개" 상태 유지 안내를
  README에 명기
- 이미지 용량: 포스터는 웹용으로 압축(장당 300KB 이하 목표) — 모바일 데이터 환경 고려
- 오프라인/저속: 페이지 자체는 단일 HTML로 가볍게, 갤러리 이미지는 lazy loading

### 테스트

- 모바일 뷰포트(iPhone/Android 폭)에서 레이아웃 확인
- 실제 배포 URL을 폰에서 열어 확인 후 NFC 기록
- 노션 링크가 로그인 없이 열리는지 시크릿 창에서 확인
